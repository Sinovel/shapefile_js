shapefile_js
=========

This project allows user to read shapefile with Javascript. 

A live example can be found at
http://polluxll.github.com/shapefile_js/index.html

### Support shape type

+ Point
+ PointZ
+ PointM
+ MultiPoint
+ MultiPointZ
+ MultiPointM
+ Polyline
+ PolylineZ
+ PolylineM
+ Polygon
+ PolygonZ
+ PolygonM

### Resources
